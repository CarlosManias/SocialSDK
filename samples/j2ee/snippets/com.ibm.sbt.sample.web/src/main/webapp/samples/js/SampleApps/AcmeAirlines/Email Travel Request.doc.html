This sample demonstrates how the Acme Airlines sample application sends an email with an associated embedded experience. The email contains contains content with mimeType:"application/embed+json" and content which is a JSON object representing the embedded experience. The embedded experience JSON object contains a gadget property which points to the gadget URL and a context object which is the data for the embedded experience. In this this sample the gadget URL points to the Acme Airlines sample application and the context is a flight.
