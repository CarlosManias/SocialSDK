*******************************************************************************

     IBM(R) Social Business Toolkit SDK

*******************************************************************************

This readme provides important information about the IBM Social Business 
Toolkit SDK.

The SDK is a collection of APIs for the IBM Social Business
platform. The APIs enable you to build applications that access Connections
and SmartCloud services.

------------------------------------------
INSTALLING THE SOCIAL BUSINESS TOOLKIT SDK
------------------------------------------

The SDK is packaged in a .zip archive file. To install
the SDK, extract the contents of the file to a local drive, which
creates an sbtsdk directory containing the SDK contents.

To uninstall the SDK, delete the sbtsdk directory.

------------------------------------
SOCIAL BUSINESS TOOLKIT SDK CONTENTS
------------------------------------

The SDK directory structure is shown below:

sbtsdk
   readme.txt                   - This readme file
   *license*                    - Licenses files for SDK and Open Source used therein
   config                       - SDK configuration file
   doc                          - SDK documentation
      javadoc                   - Java API documentation
      jsdoc                     - JavaScript API documentation
   redist                       - SDK re-distributable files
      bin                       - Java binary files
      domino                    - Domino files (.nsf and update site)
      war                       - Web application archive files
   samples                      - SDK samples
      ear                       - Enterprise application archive packaging of the samples
      war                       - Web application archive packaging of the samples
   source                       - SDK projects source
   tomcat                       - SDK pre-configured Tomcat build
